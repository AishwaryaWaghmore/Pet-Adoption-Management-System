#  Project Overview
This project is a database-driven system designed to manage the operations of a pet adoption center.
It maintains data related to:
- Pets
- Adopters
- Shelters
- Staff
- Visits
- Medical records
- Adoption processes

The system is implemented using:
- MySQL (Database)
- Python Tkinter (GUI)
- SQL Triggers, Functions, Procedures
